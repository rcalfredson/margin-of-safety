# Margin of Safety

Margin of Safety is a small stock-screening web app for comparing public companies using fundamental-analysis metrics inspired by Benjamin Graham-style value investing.

The goal is not to produce buy/sell signals. Instead, the app helps organize valuation, balance-sheet, profitability, and data-completeness signals so that interesting companies can be flagged for deeper manual research.

## Planned Features

- Enter and save watchlists of tickers
- Fetch quote and fundamental data from a pluggable provider
- Calculate basic value metrics such as P/E, P/B, Graham Number, debt/equity, current ratio, ROE, and margins
- Compare companies in a sortable table
- Store local snapshots for later review
- Flag incomplete or stale data
- Avoid investment advice language and focus on research workflow
